Decision Matrices

In the rows, we list all the possible actions among which we are attempting to choose.

In the columns, we list all the possible states of the world among which we are uncertain which is actually the case.

Each cell is an outcome, and we place utility values for each outcome in the relevant cell.

Example matrix

State 1 (S1) State 2 (S2)
Action 1 (A1) Val(A1S1) Val(A1S2)
Action 2 (A2) Val(A2S1) Val(A2S2)

Expected utility

We can then use this matrix to determine which of the actions before us is the best.

To do this, we need to assign probabilities to the various states of affairs. The value we assign to a state of affairs represents how likely we think it is that that state of affairs will obtain.

The expected utility is a numerical representation of how good of a choice that action is compared to the other options.

EU(A1) = Val(A1S1) × P(S1) + Val(A1S2) × P(S2) + ...

After calculating the expected value for each of the actions we have to choose from, the action that gets the highest result is the one that we have most pragmatic reason to choose.
